# 🔒 Implementación de Medidas de Ciberseguridad para la Protección de Datos en Nestlé Corporativo

## 📌 Descripción
Proyecto de ciberseguridad enfocado en la protección de datos sensibles en un entorno corporativo (caso Nestlé). Incluye identificación de vulnerabilidades, análisis de riesgos y propuesta de mejora con controles técnicos y organizacionales.

## 🎯 Objetivo General
Implementar medidas de ciberseguridad para proteger los datos sensibles y reducir riesgos de acceso no autorizado.

## 🎯 Objetivos Específicos
- Identificar vulnerabilidades en la infraestructura
- Analizar riesgos de seguridad
- Diseñar controles de protección
- Implementar medidas de seguridad
- Evaluar resultados

## 🔍 Vulnerabilidades Identificadas
- Equipos expuestos sin supervisión
- Credenciales no controladas
- Falta de monitoreo en cambios de turno

## 🛠️ Propuesta de Mejora
### Hardware
- Servidor HP ProLiant
- Firewall Fortinet
- Cámaras IP
- Control biométrico
- Switches Cisco

### Software
- SIEM (Splunk)
- IDS/IPS (Snort)
- EDR (CrowdStrike)
- VPN (OpenVPN)

## 🧪 Pruebas
- Pruebas funcionales
- Pruebas de seguridad
- Simulación de ataques

## 📊 Resultados
- Mejora en control de accesos
- Reducción de riesgos
- Mayor visibilidad de eventos

## 📈 Conclusiones
La seguridad debe integrar tecnología, procesos y personas para ser efectiva.
